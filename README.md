## SP LibraryH3lp constant Contact

<p>

## Installation & Usage

**need credentials to connect to LibraryH3lp - see lh3api package on GitLab** 

```python 
#using poetry
poetry add sp-lh3-constant-contact
#or using pip
pip install sp-lh3-constant-contact
```

To run an example of the app
```python 
poetry run python
```
